🏅 Global Olympic Performance Analysis

This project presents an in-depth analysis of Olympic Games data, focusing on medal distribution, country performance, and global trends using Power BI.

🔍 Project Overview

The Olympic Games represent the pinnacle of international athletic competition. This project analyzes historical Olympic data to identify performance patterns, dominant countries, and regional trends across both Summer and Winter Games.

⚙️ Technologies Used
Power BI – Data visualization and dashboard creation
DAX – Calculated measures and aggregations
Data Modeling – Structuring datasets for analysis
📊 Key Visualizations

The dashboard includes:

Total medal counts by country (Gold, Silver, Bronze)
Comparison of Summer vs Winter Olympic performance
Top-performing countries visualization
Global medal distribution (map-based analysis)
Participation vs medal efficiency
Continent-wise and regional performance insights
🔑 Key Insights
United States leads in overall medal count across Olympic history
Countries like China and Russia show strong and rising performance trends
Norway dominates Winter Olympics medal counts
Medal distribution highlights a concentration in developed regions (North America, Europe, Asia)
Smaller nations demonstrate limited but notable achievements
🌍 Analytical Focus
Historical medal trends (1896–2022)
Country-wise dominance and performance efficiency
Regional strengths in specific sports
Global disparities in Olympic success
🎯 Conclusion

This analysis highlights how economic strength, infrastructure, and investment in sports influence Olympic success. The dashboard provides a clear, interactive way to explore global athletic performance and identify emerging trends in international sports.
